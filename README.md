# 中华SB名录

此仓库用来收集当代SB事件（公共新闻），权为当后人小说素材。

话说，自明清起来，出了很多比较有名的批判类小说。我们在学语文的时候都有读到，比如《儒林外史》，《官场现形记》，《二十年目睹之怪现状》，《孽海花》，《老残游记》。在中国小说史上留下了浓重的一笔。  
奈何本人出身理科，文学修养不如前辈。唯有摘录当代现象或事件，为后人增加小说素材，此之谓《中华SB名录》

注：因事件来源网络，无法一一核实，也许存在偏差，读者自行判断。
